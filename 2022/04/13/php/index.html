二、变量

​ PHP 脚本语言是一种弱类型语言，和其他语言不同的是变量或者常量的数据类型由程序的上下文决定。

2.1 变量的声明

​ PHP的特性之一就是它不要求在使用变量之前声明变量，当第一次给一个变量赋值时，你才创建了这个变量，变量用于存储值比如数字，文本字符串或者数组。

​ PHP 中变量必须使用一个美元符号$，后面跟着变量名来表示，使用赋值操作符（=）给一个变量赋值。

2.2 变量的释放

​ unset() 函数释放指定的变量。

​ unset($name);

2.3 变量的命名

​ 美元符号 $ 后跟变量名称。

​ @ 严格区分大小写

​ @ 字母、数字、下划线，不能以数字开头

​ $name1

​ $my_name

​ $_my_name

​ @ 不要使用 PHP 关键字作为变量名

2.4 可变变量

​ 所谓可变变量 ，就是一个变量的名，又是一个变量。一个变量的变量名可以动态的设置和使用。

2.5 变量的引用

​ 引用符号 & ，相当于让两个变量指向同一块内存区域。

2.6 变量类型

​ 在强类型的语言中如 C 语言，变量要先指定类型，然后才可以存储对应指定的数据。

​ 但是PHP是一种弱类型的语言，和其他语言不同的是，变量或者常量的数据类型由程序的上下文决定，不需要事先声明类型。

  1. bool （布尔型）

    以下值被认为是false，其他的指收市被认为是true

    • 布尔值false
    • 0
    • 浮点型 0.0
    • 空白字符串和字符串0
    • 没有成员的数组
    • NULL

  2. int 型

    ​ 存储整数

  3. float 浮点型

    ​ 存储小数

  4. string 字符串

    ​ 字符串输出：echo 字符串名（不需要加$）

    ​ 字符串是一系列的字符

    ​ 字符串可以使用单引号、双引号、定界符三种方法来定义。

    ​ 单引号：

    ​ 指定一个简单的字符串最简单的方法是使用一对单引号 ’ 括起来。

    ​ 单引号字符串中出现的变量不会被变量的值替代，即PHP不会解析单引号中的变量， 因此在定义简单字符串的时候，使用单引号会效率更高。

    ​ 双引号：

    ​ 其中的变量名会被变量值所代替，即PHP会解析双引号之间的变量。

    ​ $ 是变量的开始，变量整体需要用 { } 括起来。

    ​ 单双引号定义的字符串中输入特殊字符需要转义。

    ​ 定界符：

    ​ <<< 之后提供一个标识符开始，然后是字符串的内容最后是同样的标识符表示结束。

    ​ 注意：结束标识符之后必须接；不可以用其他的任何字符。

    ​ 标识符的命名规则：字母数字下划线，并且不能以数字开始

    ​ 定界符开始和结束后面不能有任何字符包括空白字符和注释。

9.3.1 对数据库执行一条SQL语句

说明：

  1. 对于insert，update，delete等不会返回数据的SQL语句，在执行时没有错误时将返回true

  2. 对于返回数据的SQL语句执行成功的时候会返回结果集对象可以使用操作结果集对象的函数从中获取数据

  3. MYSQLI_STORE_RESULT 和 MYSQLI_USE_RESULT 决定了mysqli client 和 server 之间取结果集的方式。

    MYSQL_STORE_RESULT: 执行sql是提取结果集返回给client，并分配内存，存储到用户程序空间中，之后 mysqli_fetch_array() 相当于是从本地取数据；而MYSQLI_USER_RESULT 方式下，mysqli_fetch_array() 每次都要向server请求结果行。

    MYSQL_USE_RESULT: 执行SQL的时候，并没有从server将结果集取回。
